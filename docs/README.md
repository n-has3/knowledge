# nmin wiki

nminの技術メモとナレッジベースをまとめたWikiです。

## 🌐 サイトURL

https://nmin70.github.io/knowledge/

## 📝 概要

日々の開発で学んだことや技術的な発見をまとめる個人的なナレッジベース。
Jekyll + GitHub Pagesで構築されています。

## 📂 構成

- `_posts/` - 記事ファイル
- `pages/` - 固定ページ
- `assets/` - スタイル・スクリプト
- `docs/` - ビルド済みファイル

## 📦 技術

- Jekyll
- GitHub Pages
- Markdown

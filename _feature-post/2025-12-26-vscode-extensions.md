Visual Studio Codeをさらに便利にする、おすすめの拡張機能を紹介します。

## 1. Prettier - Code formatter

コードの自動フォーマッター。保存時に自動でコードを整形してくれます。

**特徴：**
- 複数の言語に対応
- チーム内でコードスタイルを統一できる
- カスタマイズ可能

## 2. ESLint

JavaScriptの静的解析ツール。コーディング規約違反やバグを検出します。

**特徴：**
- リアルタイムでエラーを検出
- 自動修正機能
- カスタムルールの設定が可能

## 3. GitLens

Gitの履歴を可視化し、コードの変更履歴を簡単に確認できます。

**特徴：**
- 各行の最終変更者と変更日時を表示
- コミット履歴の視覚化
- ブランチの比較機能

## 4. Live Server

ローカル開発サーバーを起動し、変更を自動でリロードします。

**特徴：**
- ワンクリックでサーバー起動
- ホットリロード機能
- カスタムポート設定

## 5. Path Intellisense

ファイルパスの自動補完機能を提供します。

**特徴：**
- ファイル名の自動補完
- 相対パス・絶対パスに対応
- 入力ミスを防ぐ

## 6. Bracket Pair Colorizer 2

括弧をカラフルに色分けして、対応関係を分かりやすくします。

**特徴：**
- ネストレベルごとに色分け
- 対応する括弧をハイライト
- カスタムカラー設定

## 7. Auto Rename Tag

HTMLタグの名前を変更すると、対応するタグも自動で変更されます。

**特徴：**
- 開始タグと終了タグを同時に変更
- JSX/TSXにも対応
- 入力効率アップ

## 8. REST Client

VS Code内でHTTPリクエストを送信できます。

**特徴：**
- Postmanなしでテスト可能
- リクエスト履歴の保存
- 環境変数のサポート

## 9. Todo Tree

コード内のTODOコメントを一覧表示します。

**特徴：**
- TODO、FIXME、NOTEなどを検索
- サイドバーで一覧表示
- カスタムタグの追加可能

## 10. Docker

Docker関連の操作をGUIで実行できます。

**特徴：**
- コンテナの管理
- イメージのビルド・実行
- ログの表示

## まとめ

これらの拡張機能を導入することで、開発効率が大幅に向上します。
自分の開発スタイルに合わせて、必要なものを選んでインストールしてみてください。

## インストール方法

1. VS Codeを開く
2. サイドバーの拡張機能アイコンをクリック
3. 検索バーに拡張機能名を入力
4. インストールボタンをクリック

快適な開発ライフを！
